# ChanGPT 文件

ChanGPT 是「帶點禪味的 AI」——結合佛法經典與現代 AI 技術的問答與檢索服務。
問答皆附經典出處引用，並提供 OpenAI 相容的開放 API。

ChanGPT is an AI service with a touch of Zen: Buddhist Q&A with citations,
semantic search over Buddhist texts, and an OpenAI-compatible API.

## 給使用者 For users

直接使用線上應用：[app.changpt.org](https://app.changpt.org)

## 給開發者 For developers

- **[Partner API Guide](/docs/partner_api)** — 從這裡開始。OAuth 2.0 取得權杖、
  curl 快速上手、OpenAI 相容介面與 Python 客戶端範例。
- **[API Reference](/docs/api_reference)** — 完整端點參考：認證、對話串
  （threads）、串流（SSE）、建議問題與設定。

範例程式（Python client、demo scripts）可直接下載：
[`/examples/partner_api/`](pathname:///examples/partner_api/README.md)
